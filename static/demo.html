§ Demo·Live Type-Checker

Paste a module, see the verdict.

The textarea below is wired to the same nine checkers that ship in the v0.10.1 package. Paste a .furqan module, or examine an exhibit. The result returns in milliseconds, deterministic, no model in the loop.

Each verdict is derived from the CLI's exit code and stdout, the same contract the package documents. The verdict logic below is the contract you are running against.

Ready Folio 01
Verdict logic
PASS → All nine checkers ran. Zero marads, zero advisories. The module is structurally honest.
MARAD → At least one structural violation. Each marad names the checker, the diagnosis, and the minimal fix.
ADVISORY → Informational findings only. The module passes, but the compiler has notes.
PARSE ERROR → The source could not be parsed. Reported with line and column. No checker runs on a broken AST.
